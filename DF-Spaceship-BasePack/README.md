# DF Spaceship Base Pack

This is a starter resource pack for Minecraft Java Edition 1.21.11.

What is included:

- A valid `pack.mcmeta` for resource pack version `75.0`
- Three original low-poly gun item models
- One original reactor model inspired by the general silhouette of a sci-fi fusion reactor
- Custom sound events under the `dfsp` namespace
- A screen-shake post-effect template
- Notes for DiamondFire usage and third-party asset sourcing

What is not included yet:

- Imported third-party sound files
- Imported third-party 3D model conversions from generic formats into Minecraft JSON
- Server-side logic to trigger custom sounds, models, or post effects
