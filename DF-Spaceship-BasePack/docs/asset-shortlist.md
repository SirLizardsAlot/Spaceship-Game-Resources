# Asset Shortlist

These are the strongest source packs I found for a spaceship game workflow.

## Audio

- [Kenney Sci-Fi Sounds](https://kenney.nl/assets/sci-fi-sounds) - CC0 audio pack that should be your first stop for alarms, machine hums, airlock-style sweeps, computer beeps, and futuristic weapon sounds.
- [Kenney Interface Sounds](https://kenney.nl/assets/interface-sounds) - CC0 audio pack suited for buttons, panel clicks, UI confirms, and short computer interactions.
- [Kenney Impact Sounds](https://kenney.nl/assets/impact-sounds) - CC0 pack that can help with weapon impact layers and metallic hit feedback.
- [Kenney Digital Audio](https://kenney.nl/assets/digital-audio) - CC0 pack for computer chirps, console tones, and synthetic button feedback.
- [Pixabay Red Alert Klaxon Re-Creation](https://pixabay.com/sound-effects/red-alert-klaxon-re-creation-107222/) - useful as a direct alarm reference if you want a Star Trek-style red alert siren.

Suggested mapping:

- Doors and airlock: Sci-Fi Sounds + a metallic layer from Impact Sounds
- Machines and reactor loop: Sci-Fi Sounds
- Alarm: Sci-Fi Sounds or the Pixabay klaxon
- Buttons and levers: Interface Sounds + Digital Audio
- Computers: Digital Audio + Interface Sounds
- Guns: Sci-Fi Sounds + Impact Sounds

## Models and textures

- [Kenney Blaster Kit](https://kenney.nl/assets/blaster-kit) - CC0 low-poly weapon pack; good reference for alternate pistol and rifle silhouettes.
- [Kenney Modular Space Kit](https://kenney.nl/assets/modular-space-kit) - CC0 space-station modules; useful for reactor-room shapes, hallways, and airlock structure ideas.
- [Kenney Space Kit](https://www.kenney.nl/assets/space-kit) - CC0 general space props and ship pieces.
- [Kenney Space Station Kit](https://kenney.nl/assets/space-station-kit) - CC0 station props that can help with consoles, doors, and interior dressing.
- [Kenney UI Pack - Sci-Fi](https://kenney.nl/assets/ui-pack-sci-fi) - CC0 UI reference for computer panels and ship interfaces.

## Notes

- Kenney assets are generic 3D or UI assets, not pack-ready Minecraft JSON. Use Blockbench to import or rebuild selected pieces into block/item models.
- I intentionally made the included gun and reactor models original so the pack works now without license ambiguity.
- For the reactor specifically, I would keep the silhouette inspiration from Mekanism but avoid tracing or direct texture copying from the mod.
